一道前端笔试题add(2)(6)

2015-09-11 标签:   javascript

题目

最近面前端，出了如下一道题

实现以下函数
1) add(2, 6) // 8
2) add(2)(6) // 8 请考虑兼容如下调用 add(1)(3)(5)(7)(9)

第1)问没什么难度，第2)问目前为止没人答得很好。
咋一看题目很简单，其实不然。一般童鞋第2)问只能写出如下

答案1

两层调用时很简单，只需返回一个函数，构建闭包暂存x的值，但是要实现后面无限级的函数链式调用，几乎没有人答对

答案2

这里主要考的还是闭包的应用

  1. 首先需要total这个闭包变量暂存每次累加的值
  2. 要实现多级函数链式调用，同样需要返回一个函数f,并在这个函数内实现累加
  3. 由于返回的是函数f，那么在执行+运算时，默认会调用valueOf，因此我们需要Overwrite这个方法
  4. 同时，使用console.log()时会调用toString方法，因此也一并Overwrite了
